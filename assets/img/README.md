# 画像（写真）の差し込みについて

このフォルダは、後から **写真（JPG/PNG/WebP）** を追加するための置き場です。
現状のサイトは、軽量で無劣化な **SVG イラスト**（`assets/hero-bg.svg`・`assets/bg-texture.svg`）で
トップ画像と全体の地紋を表現しています。AI で生成した写真に差し替えたい場合は、以下の手順で入れ替えられます。

## ヒーロー（トップ）を写真にする

1. 横長の写真（推奨: 1600×900 以上、軽量化のため WebP/JPG）をこのフォルダに置く
   例: `assets/img/hero-photo.webp`
2. `css/styles.css` の `.hero::before` の背景を差し替える:

   ```css
   .hero::before {
     background: url("../assets/img/hero-photo.webp") no-repeat center / cover;
     opacity: 1; /* 写真は不透明でOK */
   }
   ```

3. **文字の視認性を保つため**、写真の上に暗いスクリム（覆い）を入れる。
   `.hero` には既に濃紺グラデーションが入っているので、必要なら次を追加:

   ```css
   .hero::after {
     background: linear-gradient(90deg, rgba(8,18,46,.78) 0%, rgba(8,18,46,.35) 60%, transparent 100%);
     opacity: 1;
   }
   ```

   （左にテキスト、右に写真が見えるグラデーション。左ほど濃く＝文字が読める）

## 注意（このプロジェクトの方針）

- 画像は**権利関係がクリアなもの**だけを使う（生成画像 / 自社撮影 / 適切なライセンス素材）。
- 人物が写る写真は、**実在の個人を特定できる形では使わない**（本サイトは個人情報を扱わない方針）。
- 写真は表示前に**軽量化（圧縮・適切な解像度）**してから配置する。

## AI 画像生成について

このリポジトリで作業する Claude（claude-opus-4-8）には、本環境では画像生成ツールがありません。
写真が必要な場合は、ChatGPT/DALL·E などで生成した画像をこのフォルダに追加してください。
追加後の **HTML/CSS への組み込み（レスポンシブ・スクリム・最適化）は Claude が対応**できます。
