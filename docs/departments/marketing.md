# マーケティング部（Marketing）

トライアンフの認知・ブランディング・需要創出を担い、母集団形成（採用）にも貢献する部署。
COO「じぇい」が人事部に続いて立ち上げた第二弾。オーナー日向氏がキャリアの原点で深く関わった
領域（コスモ証券メディアマーケティング部、業界初の「月額定額制手数料＝サブスクモデル」を主導）で
あり、強い関与が見込める。

## ミッション

- 「つなげたいのは、気持ちです。」のブランドを一貫して発信し、トライアンフの認知と信頼を高める。
- BtoB の受注の入り口となるリードを創出し、プロジェクト・テンバガー（年商100億）の需要面を支える。
- 採用の **母集団形成** に貢献する（人事部最大の悩みである新卒・登録スタッフの母集団を、認知・
  ブランディング側から後押し）。

## 担当範囲

やること:

- 認知拡大・ブランディング（理念「全方位的な優しさ」「サービス品質へのプライド」を訴求の核に）
- リードジェネレーション（メーカー・キャリア・商社など BtoB 顧客の入り口づくり）
- オウンドメディア／ SNS 運用（既存チャネル: X / YouTube / Instagram / Facebook / 公式 LINE）
- サービスの価値訴求（カスタマーサクセスの実績をストーリー化）
- 採用広報での母集団形成支援（人事部と連携）

やらないこと（当面）:

- 受注のクロージング（営業領域）
- 採用選考の実務（人事領域）

## 主要 KPI（たたき台 / 日向氏と詰める）

- 認知・流入（サイト訪問、SNS フォロワー／エンゲージメント）
- リード獲得数（BtoB 問い合わせ・商談化数）
- 採用母集団数（人事部との共有指標。特に新卒・登録スタッフ）

## 運用ルール

- ブランド軸「つなげたいのは、気持ちです。」から逸れる訴求はしない。
- 誇張せず、実績と人柄で信頼を積む（「期待を超える」体験を等身大で伝える）。

## 連携

- オーナー（日向氏）: マーケ出身。方針の最終承認に加え、知見の主要な供給源。
- 人事部: 母集団形成（採用広報）で密に連携。
- 営業部（今後）: リード → 受注のハンドオフ。
- 経営企画部（今後）: テンバガー戦略との整合。
- COO「じぇい」: 全体最適の観点で調整。
