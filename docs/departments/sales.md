# 営業部（Sales）

メーカー・携帯キャリア・大手商社などからの受注獲得と、既存顧客の深耕を担う部署。COO「じぇい」が
立ち上げた第四弾。マーケティング部が生んだリードを受注につなげ、プロジェクト・テンバガー
（年商100億）の売上を直接ドライブする。

## ミッション

- 新規受注の獲得と既存顧客の深耕により、トライアンフの売上を伸ばす。
- 「売って終わり」ではなく **カスタマーサクセス（LTV 最大化）** を志向し、全接点で満足度を高める
  （日向氏の事業観: カスタマーエンジニアの価値はエンドユーザーの成功に寄り添うことにある）。

## 担当範囲

やること:

- 新規受注の獲得（メーカー / キャリア / 商社からの導入・保守・リプレース案件）
- 既存顧客の深耕・関係構築（リピート、全国案件・長期 LCM への発展）
- 提案・見積、案件クロージング
- リード → 商談 → 受注のパイプライン管理（マーケ部からのハンドオフを受ける）

やらないこと:

- リード創出・認知形成（マーケティング部の領域）
- 現場の実作業（キッティング・設置・保守などのオペレーション領域）
- 採用（人事部の領域）

## 主要 KPI（たたき台 / 日向氏と詰める）

- 受注額・受注件数
- 商談化率・受注率
- 既存顧客の継続・LTV、パイプライン金額

## 運用ルール

- **「営業至上主義」に陥らない**（日向氏が最も嫌う価値観）。「俺たちが食わせている」という姿勢を
  排し、現場・他部署への敬意を持つ。
- 「この金額ならこの程度のサービスで十分」という発想を採らない。受注条件に関わらず、期待を超える
  提案・品質を前提にする。
- 受注は現場の稼働力（社員・登録スタッフ）と整合させ、人事部と連動して無理のない引き受けを行う。

## 連携

- マーケティング部: リード → 受注のハンドオフ。
- 人事部: 受注量に応じた人員・登録スタッフの手当て。
- 経営企画部: KPI・優先順位・売上計画の整合。
- オーナー（日向氏） / COO「じぇい」: 重要案件・方針の判断と全体調整。
